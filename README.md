Project Description

Medic AI is a simple healthcare web application designed to provide basic medical information and assistance. The website allows users to explore health-related content and interact with AI-based suggestions.

This project was developed using HTML, CSS, and JavaScript to create a responsive and user-friendly interface.

Features

User-friendly interface

Responsive design for different screen sizes

AI-based medical information concept

Clean and modern UI

Technologies Used

HTML

CSS

JavaScript

Project Purpose

The purpose of this project is to practice frontend web development and build an interactive healthcare-themed website.

How to Run the Project

Download or clone the repository

Open the project folder

Run the index.html file in your browser
